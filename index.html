Benvenuti sulla pagina personale di Brischetto Salvatore Andrea. Di seguito sono riportati alcuni principi generali della cyber security:

Strumenti di crittografia

  1. Confidenzialità: Assicurarsi che le informazioni siano accessibili solo a persone autorizzate. Utilizzare tecniche come la crittografia per proteggere i dati sensibili.
  2. Integrità: Garantire che le informazioni non siano alterate in modo non autorizzato o accidentale. Utilizzare firme digitali e controlli di integrità per rilevare modifiche non autorizzate ai dati.
  3. Disponibilità: Assicurarsi che le risorse informatiche siano sempre accessibili agli utenti autorizzati. Implementare misure per prevenire interruzioni, guasti o attacchi che potrebbero rendere inaccessibili i sistemi.
  4. Autenticazione: Verificare l'identità degli utenti e delle entità che accedono ai sistemi. Utilizzare password robuste, autenticazione a due fattori e altre tecniche per garantire che solo persone autorizzate possano accedere ai dati e ai servizi.
  5. Autorizzazione: Definire i livelli di accesso appropriati per gli utenti e le entità in base alle loro responsabilità e privilegi. Limitare l'accesso solo alle risorse necessarie per svolgere il proprio lavoro.
  6. Responsabilità: Ogni utente deve essere responsabile delle proprie azioni digitali. Mantenere le password segrete, evitare di condividere informazioni sensibili e utilizzare software legalmente acquisito.

La cyber security è di fondamentale importanza per proteggere le informazioni personali, aziendali e sensibili da minacce informatiche sempre più sofisticate. Utilizzando i principi sopra citati e adottando buone pratiche di sicurezza informatica, possiamo contribuire a creare un ambiente online più sicuro per tutti.

Per ulteriori informazioni sulla cyber security e sulle migliori pratiche, si consiglia di consultare siti web e risorse affidabili come organizzazioni specializzate, enti governativi e community di esperti di sicurezza informatica.

Di seguito sono riportati alcuni principali programmi e progetti che ho realizzato:

  • AppFlask

    • Questo codice Flask definisce un'applicazione web con una sola pagina.
    Quando si accede alla radice dell'applicazione, viene visualizzato il template 'index.html'.
    Se viene inviato un modulo tramite una richiesta POST, viene estratto il valore del campo username e viene restituito il template 'welcome.html' con il nome utente inserito.
    Si consiglia di creare i file HTML 'index.html' e 'welcome.html' nella stessa directory del file Python per fornire il contenuto desiderato per le pagine.

  • Cookie

    • Questo codice Flask definisce un'applicazione web con funzionalità di login/logout utilizzando le sessioni.
    Quando si accede alla radice dell'applicazione, viene controllato se l'utente è già autenticato e viene visualizzato un messaggio di benvenuto personalizzato se lo è.
    Altrimenti, viene visualizzato un modulo di login.
    Quando viene effettuato il login, l'username viene salvato nella sessione insieme all'ultimo accesso.
    È possibile effettuare il logout per rimuovere l'username e l'ultimo accesso dalla sessione.
    Si consiglia di creare il file HTML 'login.html' nella stessa directory del file Python per fornire il modulo di login personalizzato.
    Assicurati di mantenere la chiave segreta (app.secret_key) protetta e di non divulgarla pubblicamente, poiché è utilizzata per firmare la sessione e proteggere le informazioni.

  • ServiceScan

    • Questo codice è uno script Python per la scansione dei servizi su un determinato indirizzo IP.
    Si consiglia di personalizzare l'indirizzo IP del target (target_ip),in questo esempio io ho usato l'IP di Google,
    la porta di partenza (start_port),
    la porta di fine (end_port)
    in base alle tue esigenze di scansione dei servizi.

  • TestApp

    • In sintesi, questo codice Flask definisce un'applicazione web che mostra un form di accesso.
    Quando si invia il form con l'username e la password, vengono visualizzati i valori ricevuti.
    Altrimenti, viene visualizzato il form per l'inserimento dei dati di accesso.
    Si consiglia di personalizzare ulteriormente l'applicazione aggiungendo la logica di autenticazione o l'invio dei dati di accesso a un backend per la verifica delle credenziali."

  • Aut2Ways

    • Questo è il codice di un'applicazione Flask per un sistema di accesso OTP (One-Time Password).
    In sintesi, questo codice definisce un'applicazione web che implementa un sistema di accesso OTP.
    Gli utenti possono inserire un username e una password nel form di accesso.
    Se le credenziali corrispondono a un valore fisso, viene generata una OTP casuale, viene inviata all'indirizzo email specificato

  • PasswordSec

    • Questo è il codice di un'applicazione Flask che verifica se una password fornita soddisfa determinati criteri di sicurezza.
    In sintesi, questo codice Flask definisce un'applicazione web che verifica se una password fornita soddisfa i criteri di sicurezza.
    Gli utenti possono inserire una password nel form e la funzione is_secure_password() verifica se la password è sicura o meno.
    Il risultato viene restituito all'utente come messaggio sulla pagina web.

  • Secure4Key

    • Questo è il codice di un'applicazione Flask che esegue diverse operazioni di crittografia e hash.
    In sintesi, questo codice Flask definisce un'applicazione web che esegue operazioni di hashing (MD5, SHA-1, SHA-256) e crittografia (AES, RSA) su un testo fornito dall'utente.
    L'utente può selezionare l'algoritmo desiderato e visualizzare i risultati sulla pagina web.

  • TestMail

    • Questo è il codice di un'applicazione Flask che consente di inviare email utilizzando il protocollo SMTP attraverso un account Gmail.
    In sintesi, questo codice Flask definisce un'applicazione web che consente di inviare email tramite il server SMTP di Gmail.
    Gli utenti possono compilare un form con i dettagli dell'email (mittente, destinatario, soggetto e messaggio) e inviarlo.
    L'applicazione si connetterà al server SMTP di Gmail utilizzando l'account mittente specificato, invierà l'email con i dettagli forniti e restituirà un messaggio di conferma.

  • Blockchain

    • Questo codice implementa una semplice blockchain utilizzando la crittografia hash SHA-256 con due blocchi e verifica l'integrità della catena.È importante notare che questa implementazione è solo a scopo didattico e non contiene tutte le funzionalità e le sicurezze di una blockchain completa.è possibile implementare il codice all'interno dell'applicazione Flask che invia email tramite SMTP(link sopra).
    In particolare, puoi aggiungere la funzionalità della blockchain all'applicazione Flask che invia email.
    Ad esempio, puoi creare un endpoint aggiuntivo per aggiungere blocchi alla blockchain ogni volta che viene inviata un'email.

  • Scrap

    • Questo codice utilizza la libreria Selenium per automatizzare l'interazione con un browser web.
    In sintesi, il codice utilizza Selenium per aprire una pagina web, fare clic su un link di login e quindi chiudere il browser.
    Puoi personalizzare ulteriormente il codice per adattarlo alle tue esigenze specifiche, come l'automazione di altre azioni sulla pagina web o l'utilizzo di un driver per un browser diverso da Chrome.

  • BeCrypto

    • Questo codice rappresenta un'applicazione web con Flask che consente agli utenti di registrarsi e memorizza le informazioni nel database MySQL.
    L'applicazione consente agli utenti di registrarsi inviando un modulo con un username e una password.
    La password viene crittografata e quindi memorizzata nel database MySQL.
    Inoltre, i dati della tabella "accessi" vengono stampati per verifica.

  • Request

    • Questo codice utilizza il modulo requests per effettuare una richiesta GET a una pagina web specificata tramite URL.
    Utilizza anche il modulo BeautifulSoup per analizzare l'HTML della pagina e estrarre informazioni dai suoi elementi.
    Si noti che il codice può essere personalizzato per eseguire diverse operazioni di scraping e analisi sui dati HTML delle pagine web.
    Ricorda di sostituire l'URL nel codice con quello di tuo interesse
